# 🐌 Slime Tabs Lord
**Close tabs with style!** An animated slime mascot helps you manage your browser tabs.

## ✨ Features
- 🔢 **Big canvas number** in toolbar — bold, customizable, no icon clutter
- 🎨 **Animated slime** in popup that reacts to your actions
- 👓 **Boss glasses** when idle (yes, your slime is judging you)
- 🎯 **Quick tab management** — close others or close right
- 🌈 **Fully customizable colors** — make it yours
- 💤 **Auto-blinking** — it's alive! (blinks every 8-16 seconds)
- 🍽️ **Tab eating animation** — watch your tabs disappear

## 🚀 Installation
1. Go to [Firefox Add-ons](https://addons.mozilla.org)
2. Search for "Slime Tabs Lord"
3. Click "Add to Firefox"

Or install manually:
- Download the XPI file
- Open Firefox → `about:debugging`
- Click "This Firefox" → "Load Temporary Add-on"
- Select the XPI file

## 🎮 How to Use
1. Click the extension icon in your toolbar
2. See your slime mascot with tab count
3. Click buttons to close tabs:
   - 🗑️ **Close Others** — closes all tabs except current
   - ➡️ **Close Right** — closes tabs to the right
   - ⚙️ **Settings** — customize colors and behavior

## 🛠️ Settings
- **Toolbar Number Style** — color, font weight, transparent background
- **Popup Style** — background and text colors
- **Button Colors** — personalize action buttons
- **Slime Outline** — add/remove border around slime
- **Behavior** — choose if popup closes after action

## 📋 Requirements
- Firefox 115.0 or higher
- Works on Linux, Windows, and macOS
- Optimized for Firefox 148.0+
